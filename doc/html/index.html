Text properties

fontName

Name of the font used to draw the text in the node.

Type
String
Default
"Gill Sans"
fontSize

Font size of the node text in points (1/72 inch). The default formula yields the following font sizes as the node depth increases: 21, 15, 12, 10.

Type
Number
Default
max(10, round(21 / sqrt(depth + 1)))
lineHeight

Line height of the node text expressed as a multiplier of the font size.

Type
Number
Default
1.3
textAlign

Alignment of the text within the node. The value auto aligns the text left, right or center, depending on the node's orientation (center is used for the root node).

Type
Enum
Values
left | right | center | justify | auto
Default
auto
justifyMinLines

Minimum number of lines for the justify text alignment to take effect. If the total number of lines is less than this value, the text will be centered instead.

Type
Number
Default
5
hyphenate

Controls whether the node text is hyphenated.

Type
Boolean
Default
yes
maxTextWidth

Maximum text width inside nodes

Type
Number
Default
240.0
textPadX

Horizontal padding around the bounding box of the node text.

Type
Number
Default
fontSize * 1.0
textPadY

Vertical padding around the bounding box of the node text.

Type
Number
Default
fontSize * 0.45
textBaselineCorrection

Node text baseline correction factor.

Type
Number
Default
-0.2

Stroke properties

strokeWidth

Stroke width of the outline of the node shape.

Type
Number
Default
1.3

Type
Boolean
Default
no

Type
Color
Default
rgba(0, 0, 0, 0.2)
nodeShadowBlur

Node shadow blur radius.

Type
Number
Default
3.0
nodeShadowOffsX

Horizontal offset of the node shadow.

Type
Number
Default
2.5
nodeShadowOffsY

Vertical offset of the node shadow.

Type
Number
Default
2.5
textDrawShadow

Controls whether the node text casts a shadow.

Type
Boolean
Default
no

Gradient fill properties

drawGradient

Controls whether the node shape should be filled using a linear top-down gradient.

Type
Boolean
Default
no
gradientTopColor

Color of the top of the node if the node shade is filled with a gradient.

Type
Color
Default
baseColor.lighten(.12)
gradientBottomColor

Color of the bottom of the node if the node shade is filled with a gradient.

Type
Color
Default
baseColor.darken(.04)
